# xWindow

基于[Vue@3.2+][vue]的仿`Windows`窗口的组件库。[在线文档][doc]

> `xWindow`使用了诸如`expose`、`.prop`之类的api, **请确保你的`Vue`版本不低于3.2.0**。

## 兼容性
`xWindow`使用了诸如`Proxy`、`Reflect`之类的新特性，因此需要浏览器至少实现了`ES2015`标准。需要注意的是，`xWindow`**不支持IE浏览器，也没有相关的支持计划**。
