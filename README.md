# xWindow

xWindow是一个简单易用的弹窗组件库

## 安装

```bash
# npm
npm i @dongls/xwindow

# pnpm
pnpm i @dongls/xwindow
```

组件的使用可以参照[在线文档][doc]。

## 兼容性

- 浏览器至少实现了`ES2015`标准
- [Vue][vue]版本高于3.3
