# xWindow

基于[Vue][vue]的函数式弹窗组件库。

## 安装

```bash
# npm
npm i @dongls/xwindow

# pnpm
pnpm i @dongls/xwindow
```

组件的使用可以参照[在线文档][doc]。

## 兼容性

- 浏览器至少实现了`ES2015`标准
- `Vue`版本高于3.3
